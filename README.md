# Cat-Knight

A 2D Metroidvania-style game.

---

## 🎮 Gameplay

* 使用方向鍵進行上下左右移動
* `X`：輕攻擊
* `Z`：重攻擊
* `C`：翻滾（可用來躲避傷害）
* `↑ + ↓`：向下俯衝攻擊

遊戲以地圖探索與 Boss 戰為核心玩法，屬於類銀河戰士惡魔城（Metroidvania）風格。

目前遊戲尚未完成，尚未實作最終 Boss。

---

## ✨ Features

* 自行實作**碰撞系統（Collision System）**
* 支援多地圖之間的**切換與連接（Map Transition）**
* 實作基礎的**敵人行為模式（Enemy AI）**
* 類銀河戰士惡魔城探索式地圖設計

### 🔧 Technical Highlights

* 對碰撞系統進行優化，提高運行效率與穩定性

## 🛠️ Tech Stack

* **Language**: Python
* **Library**: Pygame
* **Game Engine**: None (Built from scratch)

---

## ▶️ How to Run

### Option 1: Run Executable (Recommended)

1. 確認以下檔案位於同一資料夾：

   * `game.exe`
   * `game.py`
   * `Role.py`
   * `neko.py`
   * `img/`
   * `map/`
   * `ttf/`

2. 直接點擊 `game.exe` 即可開始遊戲

---

### Option 2: Run from Source

1. 安裝 Python 3.x

2. 安裝套件：

```bash
pip install pygame
```

3. 執行遊戲：

```bash
python game.py
```

> 若無法執行，請確認資源資料夾（img, map, ttf）與程式在同一目錄
